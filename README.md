## Projects

### Rock–Paper–Scissors with Bayesian Learning
A simulation of an adaptive agent that learns an opponent’s behavior:

- Opponent strategy modeled as a **Markov chain (transition matrix)**
- Learning agent applies **Bayesian updating** to estimate transition probabilities
- Predicts future moves and selects optimal counter-actions
- Simulated over **1000 rounds** with performance tracking
- Outputs include:
  - Learned transition matrix
  - Score evolution over time

## Technical Highlights

- Probabilistic modeling and simulation
- Online learning from sequential data
- Strategy optimization based on predictions
- Data visualization using **matplotlib**
